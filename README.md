A health-focused productivity app that integrates with ActivityWatch to monitor screen time and enforce the 20-20-20 rule using AI-powered break suggestions.
**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
